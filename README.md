# End-to-End MLOps Pipeline with Drift Detection

Production-grade MLOps system for customer churn prediction using large-scale synthetic data.

## Features
- Faker-generated dataset (200k+ rows)
- Feature engineering & risk profiling
- MLflow experiment tracking
- FastAPI inference service
- Data drift detection (KS test)
- Automated retraining with GitHub Actions
- Cloud deployment on Render/Railway (no Docker)

## Tech Stack
Python, FastAPI, MLflow, GitHub Actions, Faker, Scikit-learn
